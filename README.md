# InterviewOS — Landing Page

Landing page for the [InterviewOS](https://github.com/hiwarkhedeprasad/cluely-killer-interview-platform) AI interview platform. Single file, no dependencies, no build step.

## Stack

- Pure HTML, CSS, JavaScript
- Fonts via Google Fonts (Syne, Fraunces, JetBrains Mono)
- No frameworks. No bundlers.

## Run it

Just open the file:

```bash
open index.html
```

Or serve it locally:

```bash
npx serve .
# or
python3 -m http.server
```

## What's inside `index.html`

- Animated hero with live typewriter transcript
- Bento feature grid with mouse-tracking glow
- Agent profile cards with voice bar equalizers
- Interactive "How it works" step panel
- Custom cursor with ring follower
- Scroll-triggered reveal animations
- Fully responsive
